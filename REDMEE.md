# 📈 Quant Strategies

Progetto Python per analisi e strategie quantitative sui mercati finanziari.  
Gestione delle dipendenze tramite **Poetry** e supporto per esecuzione/debug in **VS Code**.

---

## 🚀 Requisiti

- **Python 3.13+**
- **Poetry** installato globalmente  
  👉 Installa Poetry (se non lo hai già):
```bash
curl -sSL https://install.python-poetry.org | python3 -
  poetry --version


📦 Aggiornare dipendenze

Aggiorna tutte le librerie:
```bash
poetry update
